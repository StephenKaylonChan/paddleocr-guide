# /end 命令

每日开发结束时使用，保存当天进度。

## 执行流程

### Step 1: 总结今日工作

列出今日完成的主要任务。

### Step 2: 更新 CURRENT.md

更新 `docs/ai-context/CURRENT.md`：

1. 更新「最近完成」表格
2. 添加今日总结到「当前任务」
3. 规划明日任务到「下一步」

### Step 3: 检查是否需要归档

如果 CURRENT.md 内容超过约 1000 tokens：
- 提醒用户考虑运行 `/weekly` 进行归档

### Step 4: 输出确认

```
✅ 今日进度已保存

## 今日完成
- [任务1]
- [任务2]

## 明日计划
- [任务1]
- [任务2]

📅 [当前日期]
```

## 使用示例

```
/end
```

## 注意事项

- 每天结束开发前使用
- 确保「下一步」清晰可执行
- 为下次 /start 提供足够上下文
